# Vivify — documents

ドキュメントに生命を吹き込む仕組み。
指摘を受けたらルール化し、知識を蓄積するメタ学習のフレームワーク。

## 導入方法

- [templates/](templates/) 内のファイルをプロジェクトに取り込む
   - [CLAUDE.md](templates/CLAUDE.md) → プロジェクトルートの `CLAUDE.md` に取り込む
   - [docs/index.md](templates/docs/index.md) → `docs/index.md` として配置する
   - [README.md](templates/README.md) → プロジェクトの `README.md` に取り込む

具体的な使用例は [examples/](examples/) を参照。
